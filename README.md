# README #

Lynda Video Downloader.

### What is this repository for? ###

* This software can download any Lynda.com course
* Version 1
* [**Lynda**](https://www.lynda.com) is an online video library to help train learners for various skills

### Dependencies ###

* Python 2.7 or above
* BeautifulSoup4
* lxml
* youtube-dl
